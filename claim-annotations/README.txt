------ Claim Annotations ------

Sentences from LiveJournal weblogs and Wikipedia talk pages annotated for opinionated claims.

train/test

1988/202 livejournal sentences
1997/200 wikipedia sentences

For more information regarding annotation see:

http://www.cs.columbia.edu/~sara/publications/icsc2012claim.pdf

------- DATA ------

Format: The sentence is in one file and the gold label is in the second file on the corresponding line.

train and test folders as used in paper:

livejournal_text.csv - livejournal sentences
livejournal_claim.csv - livejournal claim labels
livejournal_annotation.csv - livejournal phrase level sentiment annotations (only available for training)

wikipedia_claim.csv - wikipedia sentences
wikipedia_text.csv - wikipedia claim labels
wikipedia_annotation.csv - wikipedia phrase level sentiment annotations (only available for training)
